# Security Policy

## Supported Versions

Only the latest released version of envault receives security updates.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately through GitHub's [private vulnerability reporting](https://github.com/fmilioni/envault/security/advisories/new).

You can expect an initial response within 7 days. Once a fix is available, a
coordinated disclosure and a patched release will follow.
